⚙️ PASSO 2: PREPARAR O AMBIENTE (5 minutos)

Certifique-se que tem instalado:

   ✅ Node.js 16+ (https://nodejs.org/)
      Verificar: node --version

   ✅ PostgreSQL 12+ (https://www.postgresql.org/download/)
      Verificar: psql --version

   ✅ Navegador moderno (Chrome, Firefox, Safari, Edge)


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


🎯 PASSO 3: INSTALAR E RODAR (10 minutos)

Abra o terminal e execute estes comandos:

   # 1. Criar banco de dados
   CREATE DATABASE ted_controle;  (no PostgreSQL)

   # 2. Instalar backend
   cd backend
   npm install

   # 3. Instalar frontend  
   cd frontend
   npm install

   # 4. Rodar backend (novo terminal)
   cd backend
   npm run dev
   # Você verá: 🚀 Servidor rodando na porta 5000

   # 5. Rodar frontend (novo terminal)
   cd frontend
   npm run dev
   # Você verá: Local: http://localhost:3000/

   # 6. Abrir navegador
   http://localhost:3000

   # 7. Cadastrar-se e começar a usar!
