🎯 PASSO 3: INSTALAR E RODAR (10 minutos)

Abra o terminal e execute estes comandos:

   # 1. Criar banco de dados
   CREATE DATABASE ted_controle;  (no PostgreSQL)

   # 2. Instalar backend
   cd backend
   npm install

   # 3. Instalar frontend  
   cd frontend
   npm install

   # 4. Rodar backend (novo terminal)
   cd backend
   npm run dev
   # Você verá: 🚀 Servidor rodando na porta 5000

   # 5. Rodar frontend (novo terminal)
   cd frontend
   npm run dev
   # Você verá: Local: http://localhost:3000/

   # 6. Abrir navegador
   http://localhost:3000

   # 7. Cadastrar-se e começar a usar!

🆘 PROBLEMAS?

Problema: "Erro ao conectar banco de dados"
Solução: Certificar que criou o banco: CREATE DATABASE ted_controle;

Problema: "Porta 5000 já está em uso"
Solução: Editar backend/.env e mudar PORT=5001

Problema: "npm command not found"
Solução: Instalar Node.js de novo

Problema: "Cannot find module"
Solução: Excluir pasta node_modules e fazer npm install novamente
